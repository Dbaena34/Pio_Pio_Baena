INSTRUCCIONES DE INSTALACIÓN
========================================

PRIMERA VEZ (Solo una vez):
------------------------------------

1. Haz doble clic en "instalar.bat"

2. El instalador hará lo siguiente automáticamente:
   - Descargar Python 3.12 (si no lo tienes)
   - Instalar Python
   - Instalar todas las librerías necesarias
   - Crear un acceso directo en tu escritorio

3. Espera a que termine (puede tardar 5-10 minutos)

4. Cuando veas "INSTALACIÓN COMPLETADA", ya está listo


CÓMO USAR LA APLICACIÓN
========================================

Opción 1 (Recomendada):
------------------------------------
- Haz doble clic en el acceso directo del escritorio:
  "Pío Pío Baena"


Opción 2:
------------------------------------
- Haz doble clic en "iniciar.bat" en esta carpeta


IMPORTANTE:
------------------------------------
✓ La aplicación se abrirá en tu navegador (Chrome, Edge, etc.)

✓ NO cierres la ventana negra (CMD) que aparece
  mientras estés usando la aplicación

✓ Para cerrar la aplicación:
  - Cierra la pestaña del navegador
  - Cierra la ventana negra (CMD)
  O simplemente presiona Ctrl+C en la ventana negra


SOLUCIÓN DE PROBLEMAS
========================================

Problema: No se abre en el navegador
------------------------------------
Solución:
1. Abre tu navegador manualmente
2. Escribe en la barra de direcciones:
   http://localhost:8501


Problema: Error al iniciar
------------------------------------
Solución:
1. Ejecuta "instalar.bat" nuevamente
2. Si persiste, contacta al soporte técnico


Problema: La aplicación va lenta
------------------------------------
Solución:
1. Cierra otras aplicaciones pesadas
2. Reinicia la aplicación


ARCHIVOS IMPORTANTES
========================================

NO BORRES ESTOS ARCHIVOS:
------------------------------------
- app.py                 (Archivo principal)
- requirements.txt       (Lista de dependencias)
- instalar.bat          (Instalador)
- iniciar.bat           (Ejecutor)
- Carpeta "modules"     (Código de la aplicación)
- Carpeta "data"        (Base de datos)
- Carpeta "images"      (Imágenes del sistema)


PUEDES BORRAR:
------------------------------------
- Carpeta "temp" (si existe, son archivos temporales)


BASE DE DATOS
========================================

Ubicación: data/granja.db

- Este archivo contiene TODOS tus datos
- Haz copias de seguridad regularmente
- Para hacer backup: copia el archivo "granja.db"
  a una ubicación segura (USB, OneDrive, etc.)


ACTUALIZAR LA APLICACIÓN
========================================

Si recibes una nueva versión:
------------------------------------
1. Haz backup de "data/granja.db"
2. Reemplaza todos los archivos EXCEPTO la carpeta "data"
3. Ejecuta "instalar.bat" nuevamente
4. ¡Listo!
